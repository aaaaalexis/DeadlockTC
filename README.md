# Deadlock 台灣正體中文轉換工具

自動轉換 Deadlock 官方簡體中文翻譯為台灣正體中文。

> [!IMPORTANT]
> 每次遊戲更新後要重新跑一次

## 使用

直接執行 DeadlockTC.py 或到 [Releases](https://github.com/aaaaalexis/DeadlockTC/releases) 下載編譯好的執行檔。
